# AWS: _VPC_

**Virtual Private Cloud (VPC)** ทำหน้าที่ในการจัดการ Network แทบจะทั้งหมดภายใน AWS ในรูปแบบเสมือนจริง(Virtual)
โดยที่เราไม่ต้องไปยุ่งกับ Physical layer
